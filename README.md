pyqqweibo
=========

腾讯微博 API Python 绑定, 基于 Twitter API 绑定库 tweepy 改写. 目前在不断完善中.

已经完成功能
------------

相关例子请参考 ''example.py''.

* 授权
* 所有 API json 请求返回
* 部分数据 model
